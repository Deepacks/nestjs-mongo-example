## Description

[Nest](https://github.com/nestjs/nest) example with mongoose integration.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
